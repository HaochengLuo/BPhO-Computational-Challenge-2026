# BPhO-Computational-Challenge-2026

## Introduction

This repository contains our code for the **British Physics Olympiad (BPhO) Computational Challenge 2026**.

The 2026 challenge theme is **Quantum Mechanics**. The models will be built in **Python** and will focus on simulation, plotting, animation, and visualisation of the challenge tasks.

## Task Overview

1. Random walk
2. Brownian motion
3. Planck black-body spectrum and Einstein heat-capacity model
4. Photoelectric effect
5. Hydrogen emission spectrum and Bohr model
6. Electron diffraction
7. Particle in a box
8. Quantum cryptography mismatch probabilities
9. Compton scattering
10. Hydrogenic orbitals

Solutions will be added as the challenge work progresses.
