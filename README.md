# BPhO-Computational-Challenge-2026

## Introduction

This repository contains my code for the **British Physics Olympiad (BPhO) Computational Challenge 2026**.

The 2026 challenge theme is **Quantum Mechanics**. The models are built in **Python** and focus on simulation, plotting, animation, and visualisation of the challenge tasks.

## Task Overview

1. Random walk
2. Brownian motion
3. Planck black-body spectrum and Einstein heat-capacity model
4. Photoelectric effect
5. Hydrogen emission spectrum and Bohr model
6. Electron diffraction
7. Particle in a box
8. Quantum cryptography mismatch probabilities
9. Compton scattering
10. Hydrogenic orbitals

## Solutions

- `task1/main.py` - random walk with `N` steps of size `s`.
- `task2/main.py` - Brownian motion with random small particles and momentum-conserving collisions with a large particle.
- `task3/main.py` - Planck black-body spectra and Einstein heat capacity model.
- `task4/main.py` - photoelectric stopping voltage for different metals.
- `task5/main.py` - hydrogen emission spectrum from Bohr model energy levels.
- `task6/main.py` - electron diffraction rings and straight-line voltage check.
